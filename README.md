# meteor-handson
meteor.jsのハンズオン用

## meteor.jsのインストール
```
curl https://install.meteor.com/ | sh
```

## versionチェック
```
meteor --version
```

## アプリ作成
```
meteor create sample-app --release 1.2
```

## アプリ起動
```
meteor
```
